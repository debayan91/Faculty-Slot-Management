# **App Name**: React Start

## Core Features:

- Homepage: A landing page with a brief introduction and navigation options.
- Navigation: Basic navigation to different sections/pages.
- Theming Support: Toggle between light and dark modes using a React context.

## Style Guidelines:

- Dark color scheme.
- Primary color: Indigo (#4F669B), suggesting calm efficiency.
- Background color: Near-black (#12141A). A dark surface allows content to pop.
- Accent color: Violet (#7B5ABF). Used for highlights.
- Body and headline font: 'Inter', sans-serif.
- Simple, outline-style icons for a modern look.
- Clean, minimalist layout with ample white space.
- Subtle transitions and animations for user interactions.
